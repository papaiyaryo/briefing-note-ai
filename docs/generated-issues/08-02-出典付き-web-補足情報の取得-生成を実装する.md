# [Phase 8] 出典付き Web 補足情報の取得・生成を実装する

## Phase

Phase 8: Web補足機能

## Priority

Low

## Labels

- phase:8
- post-mvp
- backend
- llm

## Goal

公式サイト等から取得した情報を出典付きで補足候補として生成する。

## Scope

- 企業名から補足対象ページを取得する
- 補足候補に出典 URL と取得日を付ける
- 信頼度や要確認表示を付与する

## Out of Scope

- ユーザーに無断で Markdown へ自動統合
- 非公式情報の大量収集

## Acceptance Criteria

- [ ] 補足情報に出典が必ず付く
- [ ] 紙メモ由来情報と混ざらない
- [ ] 不確実情報は要確認として扱われる

## Dependencies

Depends on #08-01。

## Notes

検索や取得の最新仕様は実装時に確認する。
