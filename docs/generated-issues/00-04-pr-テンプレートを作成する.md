# [Phase 0] PR テンプレートを作成する

## Phase

Phase 0: 設計・開発準備

## Priority

Medium

## Labels

- phase:0
- mvp
- docs
- setup

## Goal

PR ごとに仕様・テスト・スコープ外を確認できるテンプレートを用意する。

## Scope

- .github/pull_request_template.md を作成する
- Summary / Changes / Test Plan / Out of Scope / Follow-up Tasks を含める
- 秘密情報や MVP 外実装の確認項目を含める

## Out of Scope

- GitHub Actions の実装
- Issue テンプレートの整備

## Acceptance Criteria

- [ ] PR テンプレートが .github 配下に配置されている
- [ ] AGENTS.md の PR Rules と整合している
- [ ] レビュー時の確認項目が明記されている

## Dependencies

None

## Notes

実装フェーズ開始前に用意すると PR 粒度が安定する。
