# [Phase 8] Web 補足機能の仕様と取得方針を整理する

## Phase

Phase 8: Web補足機能

## Priority

Low

## Labels

- phase:8

## Goal

企業情報を Web から補う際の対象・出典・信頼度の方針を決める。

## Scope

- 公式サイト・採用ページを優先する方針を定義する
- 取得日・出典 URL の扱いを定義する
- Web 補足をユーザー確認後に採用する流れを整理する

## Out of Scope

- Web スクレイピング実装
- 自動企業データベース作成

## Acceptance Criteria

- [ ] Web 由来情報と紙メモ由来情報の分離方針がある
- [ ] 出典と取得日を残す仕様がある
- [ ] MVP 外機能として整理されている

Depends on MVP and OpenAI design as needed。
