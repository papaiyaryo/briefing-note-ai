# [Phase 3] 企業名・イベント情報入力 UI を実装する

## Phase

Phase 3: MVP UI

## Priority

Medium

## Labels

- phase:3
- mvp
- frontend

## Goal

Markdown 生成に使う企業名や説明会情報をユーザーが補えるようにする。

## Scope

- 企業名入力欄を追加する
- 説明会日・イベント名などの任意入力欄を追加する
- 未入力時の扱いを UI に反映する

## Out of Scope

- 企業データベース化
- 自動企業名補完

## Acceptance Criteria

- [ ] 企業名が未入力でもフローが進められる
- [ ] 入力値が Markdown 生成の状態に渡せる
- [ ] 個人情報を不要に要求しない

## Dependencies

Depends on #03-01。

## Notes

企業名はファイル名生成にも利用する。
