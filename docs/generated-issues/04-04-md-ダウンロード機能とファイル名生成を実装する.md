# [Phase 4] .md ダウンロード機能とファイル名生成を実装する

## Phase

Phase 4: Markdown生成・ダミーOCR

## Priority

High

## Labels

- phase:4
- mvp
- frontend

## Goal

編集済み Markdown をローカルの .md ファイルとして保存できるようにする。

## Scope

- Markdown 文字列から Blob を作る
- ブラウザダウンロードを実装する
- 企業名または日時ベースのファイル名を生成する

## Out of Scope

- Google Drive 保存
- サーバー側保存

## Acceptance Criteria

- [ ] 編集後の Markdown が .md としてダウンロードできる
- [ ] 企業名がある場合はファイル名に反映される
- [ ] 企業名がない場合は日時ベースで安全なファイル名になる

## Dependencies

Depends on #03-05 and #04-02。

## Notes

ファイル名は OS で問題になりやすい文字を sanitize する。
