## Goal

- カラー・タイポグラフィ・余白の方針を整理する

- docs/ui-spec.md に反映する

## Out of Scope

- 本格的なデザインシステム構築
- Storybook 導入

## Acceptance Criteria

- [ ] MVP で使う基本 UI ルールが docs にある
- [ ] アクセシビリティ上の最低限の配慮が記載されている
- [ ] Tailwind 実装で参照できる粒度になっている
