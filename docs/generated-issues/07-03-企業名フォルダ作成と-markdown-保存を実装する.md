# [Phase 7] 企業名フォルダ作成と Markdown 保存を実装する

## Phase

Phase 7: Google Drive連携

## Priority

Low

## Labels

- phase:7
- post-mvp
- google-drive
- backend

## Goal

編集済み Markdown を企業名フォルダに保存できるようにする。

## Scope

- 企業名フォルダの作成または検索を実装する
- Markdown ファイルを Drive に保存する
- 保存完了リンクを UI に返す

## Out of Scope

- 元画像保存
- 複数端末同期

## Acceptance Criteria

- [ ] 企業名ごとに保存先を整理できる
- [ ] Markdown ファイル名が安全に生成される
- [ ] 保存完了後に Drive リンクが表示される

## Dependencies

Depends on #07-02 and #04-04。

## Notes

企業名不明時のフォルダ名ルールを決める。
