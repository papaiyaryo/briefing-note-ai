# [Phase 2] Figma で MVP UI デザインを作成する

## Phase

Phase 2: Figma / UI設計

## Priority

High

## Labels

- phase:2
- mvp
- figma
- frontend

## Goal

アップロードから Markdown ダウンロードまで迷わず進める MVP UI を Figma 上で設計する。

## Scope

- アップロード画面を設計する
- OCR 結果確認画面を設計する
- Markdown 編集・プレビュー画面を設計する

## Out of Scope

- React 実装
- Post-MVP 画面の詳細設計

## Acceptance Criteria

- [ ] MVP の主要フローが Figma 上に存在する
- [ ] 空状態・エラー・ローディングの方針が含まれる
- [ ] MVP 外機能が主要導線に混ざっていない

## Dependencies

Depends on #02-01。

## Notes

就活中に素早く使える静かな UI を優先する。
