# [Phase 3] 画像アップロード UI を実装する

## Phase

Phase 3: MVP UI

## Priority

High

## Labels

- phase:3
- mvp
- frontend

## Goal

紙メモ画像を選択し、ファイル名とプレビューを確認できる UI を作る。

## Scope

- 画像選択 UI を実装する
- 対応形式とサイズ制限の表示を追加する
- 選択済み画像のプレビューを表示する

## Out of Scope

- OCR 実行処理
- 複数画像アップロード

## Acceptance Criteria

- [ ] png / jpg / jpeg / webp の選択を想定した UI である
- [ ] 未選択・選択済み・不正形式の状態が表示される
- [ ] アップロード画像の内容をログ出力しない

## Dependencies

Depends on #03-01。

## Notes

1 回の処理は 1 画像に限定する。
