# [Phase 8] Web 補足の確認 UI と Markdown 統合を実装する

## Phase

Phase 8: Web補足機能

## Priority

Low

## Labels

- phase:8
- post-mvp
- frontend
- llm

## Goal

ユーザーが採用した Web 補足だけを Markdown に統合できるようにする。

## Scope

- Web 補足候補を一覧表示する
- 採用・却下の操作を実装する
- 出典付きで Markdown に追記する

## Out of Scope

- 自動一括採用
- 企業比較機能

## Acceptance Criteria

- [ ] ユーザー確認なしに補足が本文へ混ざらない
- [ ] 出典 URL と取得日が Markdown に残る
- [ ] 信頼度または要確認表示が確認できる

## Dependencies

Depends on #08-02。
