# Product

## 概要

briefing-note-ai は、紙の企業説明会メモを OCR と LLM で構造化 Markdown に変換し、就職活動の企業研究、エントリーシート作成、面接対策に再利用できるようにする Web アプリです。

## 解決したい課題

企業説明会では、配布資料や話の流れに沿って紙にメモを取ることが多い一方で、後から見返すと情報が散らばり、企業比較や ES・面接準備に使いにくくなります。

このアプリは、手書きメモをデジタル化するだけでなく、就活で使いやすい観点に整理し直すことを目的にします。

## 想定ユーザー

- 企業説明会や合同説明会に参加する就活生
- 紙のメモを後から企業研究に活用したい学生
- ES や面接前に企業ごとの情報を短時間で見返したい学生

## コア価値

- 紙メモをアップロードするだけで、OCR 結果を確認できる
- OCR 結果をもとに、企業研究向けの Markdown に変換できる
- 生成された Markdown を編集し、就活用の個人ナレッジとして保存できる
- `.md` としてダウンロードし、任意のノートアプリや Git 管理に持ち出せる

## MVP

MVP では、以下に絞って実装します。

- 画像アップロード
- OCR 結果表示
- Markdown 生成
- Markdown 編集
- `.md` ダウンロード

## 将来的な機能

- OpenAI Vision API による画像理解と高精度な構造化
- Google Drive への保存
- Web 補足による企業情報の追記
- 複数企業の比較
- 面接前復習モード

## 成功指標

- ユーザーが紙メモから Markdown を生成し、編集後にダウンロードできる
- 生成 Markdown が企業研究、ES、面接対策に再利用しやすい構造になっている
- MVP の操作が、アップロードからダウンロードまで迷わず完了できる
